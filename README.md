# Amazon UI Replica

A responsive Amazon homepage clone built using HTML and CSS. This project replicates the Amazon India UI layout including navbar, hero section, product grid, and footer styling.

## Features

- **Responsive Navigation Bar**: Includes Amazon logo, location selector, search bar with category dropdown, sign-in, returns/orders, and cart
- **Secondary Navigation Panel**: Menu with "All", deals, customer service, and other links
- **Hero Section**: Promotional banner with gradient background
- **Product Grid**: 8 product categories with colorful gradient placeholders
  - Clothes
  - Health & Personal Care
  - Furniture
  - Electronics
  - Beauty picks
  - Pet Care
  - New Arrivals in Toys
  - Discover Fashion Trends
- **Footer**: Multi-section footer with links organized by category
- **Fully Responsive**: Adapts to desktop, tablet, and mobile screen sizes

## Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Embedded SVG for Amazon logo

## Responsive Breakpoints

- **Desktop**: 1200px and above (4 columns)
- **Tablet**: 768px - 1199px (2 columns)
- **Mobile**: Below 768px (1 column, stacked layout)

## How to Use

1. Clone the repository
2. Open `index.html` in your web browser
3. No build process or dependencies required!
